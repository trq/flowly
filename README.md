# Flowly

Flowly is an agent to help manage your budget.

## Project Structure

- `web/` - Frontend app (Vite + React)
- `api/` - Backend app workspace (placeholder for now)

## Requirements

- [Bun](https://bun.sh/) 1.3.5+

## Workspace Commands

Run these from the project root:

- `bun install` - Install dependencies
- `bun run dev` - Start the web app
- `bun run dev:web` - Start only the web app
- `bun run dev:api` - Start only the API app (placeholder today)
- `bun run build` - Build all workspaces
- `bun run build:web` - Build only the web app
